Getting started

How to get from “fresh VM” → “working demo toolchain”.

Docs home Skyforge home Platform status

1) Log in once (Skyforge session)

  1. Open Skyforge home.
  2. Click Sign In to authenticate via the Skyforge SSO (Dex + LDAP).
  3. After sign-in, use the SSO links like /netbox/, /nautobot/, /git/, and /minio-console/ (object storage console).

2) Validate the stack

Check the live status JSON:

curl -k https://localhost/status/summary | head

And verify pods:

kubectl -n skyforge get pods

3) Useful unauthenticated links

  • File Drop (anonymous HTTPS upload/download)
  • Coder Workspace (sign-in required)
  • Docs

4) Secrets & config

Internal credentials are stored under ./secrets/ (gitignored). Update files and re-apply the secrets overlay as needed.
